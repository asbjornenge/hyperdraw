# hyperdraw

Hyperdraw is a shared cli pixel canvas. It was made to show off the capabilities of [hyperswarm](https://github.com/asbjornenge/hyperswarm).

## Install

```sh
npm install -g @asbjornenge/hyperdraw
```

## Use

```sh
hyperdraw --color green
```
